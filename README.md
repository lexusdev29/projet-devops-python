# Projet DevOps Python

Ce projet est une application Flask simple (API + interface web) utilisée pour construire une chaîne DevOps complète :

- Gestion de code avec Git
- Conteneurisation Docker
- Pipeline CI/CD (GitHub Actions ou GitLab CI)
- Déploiement avec Docker Compose

## Lancer le projet en local

```bash
pip install -r requirements.txt ## Pour installer tous les packages nécessaires au projet d'un coup.
python run.py ## Pour lancer sur serveur web mai aussi importer l'application Flask.
